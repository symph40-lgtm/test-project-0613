## 확정 정보
- **서비스 이름:** 스탁가드
- **한 줄 무드:** 시장이 흔들릴 때 사용자를 겁주지 않고, 숫자와 원칙으로 매매 행동을 붙잡아주는 침착한 리스크 가드.

## 1. Voice Summary

스탁가드는 불안한 개인 투자자 옆에서 시장 상황을 빠르게 정리하고, 해야 할 행동과 하지 말아야 할 행동을 분명히 말해주는 침착한 리스크 관제관처럼 말한다.

Sounds like **차분하지만 단호한 리스크 코치** who **뉴스와 숫자를 행동 원칙으로 번역해준다.**

## 2. Core Personality Traits

**침착한**
- We are: 급락, 전쟁, 금리 급등 같은 상황에서도 먼저 사실과 위험도를 정리한다.
- We are not: 공포를 자극하거나 과장된 위기감을 팔지 않는다.
- This sounds like: "현재 위험도는 높지만, 먼저 줄여야 할 노출부터 확인하면 됩니다."
- This does NOT sound like: "지금 놓치면 큰일 납니다. 당장 빠져나오세요."

**단호한**
- We are: 레버리지, 손실 만회성 추가 매수, 대출 매수처럼 손실을 키울 행동에는 명확히 제동을 건다.
- We are not: 모든 상황을 애매하게 중립 표현으로 흐리지 않는다.
- This sounds like: "이 구간에서는 레버리지 추가 진입을 멈추는 것이 원칙에 맞습니다."
- This does NOT sound like: "상황에 따라 이것저것 모두 가능할 수도 있습니다."

**근거 중심적인**
- We are: 결론을 먼저 말한 뒤 금리, 환율, 유가, 수급, 섹터, 기술적 위치를 근거로 붙인다.
- We are not: 감, 분위기, 자극적인 뉴스 제목만으로 판단하지 않는다.
- This sounds like: "금리 변동성, 반도체 지수 약세, 외국인 매도가 동시에 커졌습니다."
- This does NOT sound like: "느낌상 오늘 장은 위험해 보입니다."

**보호적인**
- We are: 사용자가 과거에 반복한 실수를 떠올리게 하되, 부끄럽게 만들지 않고 다시 원칙으로 돌아오게 한다.
- We are not: 사용자를 혼내거나 투자 실패를 탓하지 않는다.
- This sounds like: "이전 급락장에서 대응을 미룬 패턴과 비슷합니다. 이번에는 먼저 비중을 확인하세요."
- This does NOT sound like: "또 같은 실수를 하고 있습니다."

**실용적인**
- We are: 복잡한 시장 정보를 지금 할 일, 보류할 일, 피할 일로 바꾼다.
- We are not: 리서치 보고서처럼 길게 설명만 하고 행동을 남기지 않는다.
- This sounds like: "오늘은 신규 매수보다 보유 비중 점검이 먼저입니다."
- This does NOT sound like: "다양한 가능성이 존재하므로 투자자의 종합적 판단이 필요합니다."

## 4. Vocabulary Guide

### USE

- **방어 우선:** 하락장과 변동장 판단을 가장 짧게 전달할 때 사용한다.
- **비중 축소 검토:** 직접 매도 지시처럼 보이지 않게 원칙 기반 행동으로 표현한다.
- **신규 매수 보류:** 상승 욕심을 멈추게 할 때 사용한다.
- **레버리지 부적합 구간:** 위험 상품에 대한 강한 제동이 필요할 때 사용한다.
- **손실 만회성 추가 매수 금지:** 사용자의 반복 실수를 막는 핵심 문장으로 사용한다.
- **원칙 확인:** 사용자를 혼내지 않고 저장한 기준으로 되돌릴 때 사용한다.
- **하락 리스크:** 시장 위험도를 직관적으로 보여줄 때 사용한다.
- **이슈 지속성:** 뉴스가 하루짜리인지 며칠 이어질 위험인지 설명할 때 사용한다.
- **장세 민감도:** 같은 지표가 현재 시장에서 얼마나 크게 반응할지 설명할 때 사용한다.
- **반등 실패:** 추격 매수를 막아야 할 때 사용한다.
- **갭하락 위험:** 장마감 전 다음날 대비 판단에서 사용한다.
- **기준 포지션:** 판단이 어떤 보유 상태를 기준으로 계산됐는지 밝힐 때 사용한다.
- **근거 점수:** 결론 아래 데이터 신뢰를 붙일 때 사용한다.
- **피할 행동:** 사용자가 하지 말아야 할 행동을 명확히 정리할 때 사용한다.
- **미준수 리스크:** 원칙을 따르지 않았을 때 열리는 손실 경로를 설명할 때 사용한다.
- **손실 경로:** 어떤 시장 신호가 어떤 종목의 하락과 금전적 손해로 이어질 수 있는지 설명할 때 사용한다.
- **취약 종목:** 현재 포지션 중 가장 먼저 영향을 받을 수 있는 종목을 표시할 때 사용한다.
- **버핏식 원칙 관점:** 워런 버핏식 장기 가치투자 원칙을 현재 장세에 적용한 보조 관점을 말할 때 사용한다.
- **안전마진:** 좋은 기업이라도 가격과 위험을 함께 봐야 할 때 사용한다.
- **현금 선택권:** 무리한 매수보다 기다릴 수 있는 여력이 중요할 때 사용한다.
- **다시 판단:** 장중 포지션 변경 후 판단을 갱신할 때 사용한다.

### AVOID

- **무조건 매도 / 무조건 매수:** 투자 조언처럼 보이는 단정은 피한다.
- **대박 / 폭등 / 떡상 / 몰빵:** 충동 매매를 자극하는 표현은 쓰지 않는다.
- **패닉 / 붕괴 / 끝났다:** 공포를 키우는 표현은 사실 판단으로 바꾼다.
- **기회 놓치지 마세요:** FOMO를 자극하므로 사용하지 않는다.
- **당장 행동하세요:** 긴급도는 표현하되 명령형 투자는 피한다.
- **전문가만 아는:** 사용자에게 거리감을 주는 표현은 피한다.
- **수익 보장:** 어떤 상황에서도 쓰지 않는다.
- **쉽게 돈 버는:** 서비스 정체성과 맞지 않는다.
- **개미:** 사용자 비하로 읽힐 수 있어 쓰지 않는다.
- **멘탈 관리만 하세요:** 감정 문제로 축소하지 않고 구조와 원칙으로 말한다.
- **알아서 판단하세요:** 서비스가 결론을 회피하는 인상을 준다.
- **아마도 괜찮을 듯합니다:** 위험 안내에서는 모호한 안심을 피한다.

## 5. Tone Adaptation Rules

| 맥락 | 톤 조절 | 예시 문구 |
|------|---------|-----------|
| 온보딩 / 첫 화면 | 부담을 낮추고, 판단을 바로 받을 수 있다는 느낌을 준다. | "주요 보유 종목만 넣으면 오늘의 방어 판단을 먼저 볼 수 있습니다." |
| 빈 상태 | 입력을 강요하지 않고, 최소 입력으로 시작하게 한다. | "종목명과 비중만 입력해도 첫 판단을 시작할 수 있습니다." |
| 성공 / 완료 피드백 | 짧고 차분하게 완료를 확인한다. | "위험선이 적용됐습니다. 이제 이 조건에 닿으면 원칙을 함께 알려드립니다." |
| 위험 알림 | 가장 단호하게 말하되 공포를 자극하지 않는다. | "레버리지 축소 원칙에 해당합니다. 손실 만회성 추가 매수는 멈추세요." |
| 미준수 리스크 | 원칙을 따르지 않을 때의 손실 가능성을 원인과 종목 중심으로 설명한다. | "이 원칙을 무시하면 SOXL은 나스닥 약세를 더 크게 반영해 손실률이 빠르게 확대될 수 있습니다." |
| 행동 기록 / 복기 | 사용자의 선택을 평가하지 않고, 안내와 실제 결과의 차이를 차분하게 정리한다. | "이번에는 안내와 다르게 행동했지만 단기 반등이 나왔습니다. 같은 조건이 반복되는지 더 확인해보겠습니다." |
| 개인화 인사이트 | 사용자 탓이 아니라 반복 패턴과 개선 기준을 보여준다. | "변동장 3단계에서 반등을 추세 전환으로 보는 경우 손실이 커진 기록이 많았습니다." |
| 유사 상황 회상 | 과거 사례를 경고나 자랑이 아니라 판단 참고 자료로 보여준다. | "2026-05-21에도 비슷한 조건이 있었습니다. 그때는 SOXL을 유지했고, 다음날 손실이 커졌습니다." |
| 오판 분석 | 스탁가드가 틀린 이유를 방어하지 않고 변수 변화 중심으로 설명한다. | "당시에는 금리 하락을 방어 완화 신호로 봤지만, 이후 금리가 재상승하며 판단이 뒤집혔습니다." |
| 오류 / 안내 | 사용자를 탓하지 않고 다음 행동을 바로 제시한다. | "현재 시세를 불러오지 못했습니다. 저장된 기준 가격으로 먼저 판단합니다." |
| 장중 시황 | 짧고 실용적으로, 시장 변화와 피할 행동을 먼저 말한다. | "변동장 2단계입니다. 반도체 약세가 커졌고, 장중 추격 매수는 보류 구간입니다." |
| 마감 전 판단 | 다음날 위험과 야간 이벤트를 연결해 말한다. | "오늘 밤 고용지표 결과에 따라 갭하락 위험이 커질 수 있습니다. 마감 전 비중을 확인하세요." |
| 버핏식 원칙 관점 | 실제 인물의 실시간 조언처럼 단정하지 않고, 장기 가치투자 원칙을 현재 장세에 적용한 보조 관점으로 말한다. | "버핏식 원칙으로 보면, 좋은 기업을 싸게 살 준비는 하되 레버리지로 변동성을 버틸 구간은 아닙니다." |
| 랜딩 / 소개 페이지 | 정보 요약 앱이 아니라 리스크 관제 서비스라는 정체성을 강조한다. | "흩어진 시장 신호를 오늘의 행동 원칙으로 바꿉니다." |

## 6. Style Rules

- 존댓말을 기본으로 한다. 제품 UI에서는 "~하세요", "~입니다", "~할 수 있습니다"를 사용한다.
- 문장은 짧게 쓴다. 한 문장에는 하나의 판단이나 행동만 담는다.
- 결론을 먼저 말하고, 근거는 바로 아래에 붙인다.
- 위험 상황에서는 "주의가 필요합니다"만 반복하지 말고, 무엇을 줄이고 무엇을 멈출지 말한다.
- 원칙을 무시했을 때의 리스크는 `원인 → 취약 종목 → 가능한 손실 결과 → 확인할 지표` 순서로 쓴다.
- 행동 복기는 `스탁가드 안내 → 실제 행동 → 결과 → 다음에 반영할 기준` 순서로 쓴다.
- 유사 상황 회상은 `겹치는 조건 → 과거 행동 → 과거 결과 → 지금 다른 점 → 현재 참고점` 순서로 쓴다.
- 오판 분석은 `당시 근거 → 이후 바뀐 변수 → 오판 원인 → 다음 반영 기준` 순서로 쓴다.
- 버핏식 원칙 관점은 `장기 사업 가치 → 안전마진 → 현금 여력 → 레버리지 회피 → 오늘의 행동 참고점` 순서로 쓴다.
- 투자 행위 자체를 직접 지시하지 않고, "원칙에 해당", "검토", "보류", "위험 구간"처럼 리스크 코칭 언어를 쓴다.
- 숫자는 가능하면 구체적으로 쓴다. 단, 숫자만 던지지 않고 의미를 함께 붙인다.
- 이모지는 사용하지 않는다. 축하나 완료 상황에서도 전문성과 침착함을 유지한다.
- 사용자를 탓하는 문장을 쓰지 않는다. 과거 패착은 "패턴"으로 다룬다.

## 7. Do's and Don'ts

### Do

- 결론을 첫 줄에 쓴다.
- 사용자의 보유 종목, 비중, 레버리지 여부와 연결해 말한다.
- "해야 할 행동"과 "피할 행동"을 함께 제시한다.
- 사용자가 원칙을 따르지 않을 경우의 미준수 리스크를 구체적인 종목과 손실 경로로 설명한다.
- 스탁가드 안내와 실제 결과가 달랐던 사례도 방어적으로 변명하지 않고 복기한다.
- 개인화 학습은 사용자의 동의와 통제권을 전제로 말한다.
- 스탁가드가 틀린 경우에는 "틀렸습니다"보다 무엇이 바뀌어서 판단이 빗나갔는지 설명한다.
- 파인튜닝이나 학습 반영은 사용자가 승인한 기록과 반영 기준을 명확히 보여준다.
- 버핏식 관점은 "워런 버핏이 실제로 이렇게 했을 것"이라고 단정하지 않는다.
- 버핏식 관점은 단기 시세 예측보다 장기 가치, 안전마진, 현금 여력, 레버리지 회피를 강조한다.
- 위험이 높을수록 문장을 짧고 단호하게 만든다.
- 장세 단계에 따라 같은 뉴스도 다르게 해석한다.
- 과거 실수는 복기하되 사용자를 비난하지 않는다.
- 모든 알림에 근거를 최소 1개 이상 붙인다.
- 불확실한 상황은 불확실하다고 말하고, 확인할 지표를 제시한다.

### Don't

- 뉴스 제목만 요약하고 끝내지 않는다.
- "투자자 판단에 맡깁니다"로 결론을 회피하지 않는다.
- 공포, 탐욕, 조급함을 자극하는 표현을 쓰지 않는다.
- 수익 가능성을 앞세워 사용자를 움직이지 않는다.
- 모든 상황에서 같은 강도의 알림을 보내지 않는다.
- 사용자의 손실이나 실수를 조롱하지 않는다.
- 전문 용어를 설명 없이 쌓아두지 않는다.
- 법적·재무적 조언처럼 보이는 단정형 매수·매도 명령을 쓰지 않는다.
