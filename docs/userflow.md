## 플로우 1 · 개장 전 방어 브리핑: "오늘 장, 일단 줄여야 하나?"

**목표 순간:** 출근 직후 3분 안에 오늘 한국장과 내 주요 보유 종목 기준으로 `방어 우선`, `유지 가능`, `신규 매수 금지`, `레버리지 부적합` 같은 결론을 얻는다.  
**상태 변화:** 정보가 흩어져 불안한 상태에서, 오늘의 대응 우선순위와 하지 말아야 할 행동이 분명한 상태로 바뀐다.

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주 행동 | 보조 |
|:--:|------|------|----------|------|
| 1 | 환영/빠른 등록 화면 | 보유 종목 입력 및 검토 | 주요 보유 종목 1~10개 입력 | 종목명, 비중, 레버리지 여부만 먼저 받음 |
| 2 | 환영/빠른 등록 화면 | 알림 연락처 설정 | 나중에 설정 또는 간단 등록 선택 | 첫 진입에서는 건너뛰기 가능 |
| 3 | 아침 브리핑 대시보드 | 개장 전 AI 종합 브리핑 | 오늘의 종합 판단 확인 | 결론을 맨 위에 표시 |
| 4 | 아침 브리핑 대시보드 | 명확한 행동 지침 | 해야 할 행동과 금지 행동 확인 | 방어/유지/축소/금지 라벨 |
| 5 | 근거 상세 화면 | 장세·섹터 판정 | 근거 점수와 주요 이슈 확인 | 금리, 환율, 유가, 미국장, 섹터 점수 |
| 6 | 포지션 관리 화면 | 보유 종목 검토 | 더 자세한 종목 정보 보완 | 가격선, 손익률, 투자 기간은 이후 입력 |

**화면**

### 1. 환영/빠른 등록 화면

```
┌────────────────────────────────────┐
│ 스탁가드                           │
│ 오늘 장에서 무엇을 줄이고          │
│ 무엇을 하지 말아야 할지 먼저 봅니다 │
│                                    │
│ 주요 보유 종목                     │
│ [ 삼성전자     ][비중 30%][일반]   │
│ [ SOXL         ][비중 20%][레버리지]│
│ [ SK하이닉스   ][비중 15%][일반]   │
│ + 종목 추가                       │
│                                    │
│ 최대 10개까지 입력 가능            │
│                                    │
│             [오늘 판단 보기]       │
└────────────────────────────────────┘
```

- **역할:** 첫 사용자가 복잡한 가입 절차보다 먼저 개인화 판단을 받을 수 있게 한다.
- **있는 정보·요소:** 종목명, 보유 비중, 레버리지 여부, 종목 추가, 오늘 판단 보기
- **주 행동:** `[오늘 판단 보기]` 선택
- **기본값:** 주요 보유 종목은 최대 10개까지 허용한다. 손익률, 현재가, 매수가는 나중에 보완해도 된다.

### 2. 아침 브리핑 대시보드

```
┌────────────────────────────────────┐
│ AI 종합 판단: 오늘은 방어 우선      │
│ 현재 장세: 변동장 3단계             │
│ 하락 리스크: 78점                  │
│                                    │
│ 해야 할 행동                       │
│ - 레버리지 비중 축소 검토           │
│ - 반도체 비중 신규 확대 보류        │
│                                    │
│ 하지 말아야 할 행동                 │
│ - 손실 만회성 추가 매수 금지        │
│ - 장초반 반등만 보고 추격 매수 금지 │
│                                    │
│ [내 종목 영향 보기] [근거 보기]     │
└────────────────────────────────────┘
```

- **역할:** 데이터보다 결론을 먼저 보여준다.
- **있는 정보·요소:** AI 종합 판단, 장세 단계, 리스크 점수, 행동, 금지, 내 종목 영향, 근거 보기
- **주 행동:** `[내 종목 영향 보기]` 또는 `[근거 보기]`

### 3. 근거 상세 화면

```
┌────────────────────────────────────┐
│ 판단 근거                           │
│ 금리 위험        82  높음           │
│ 환율 위험        71  높음           │
│ 유가 위험        68  주의           │
│ 반도체 섹터      84  취약           │
│ 이슈 지속성      높음               │
│                                    │
│ 핵심 이슈                         │
│ - 미국 기술주 약세                 │
│ - 금리 변동성 확대                 │
│ - 반도체 지수 시장 대비 부진        │
└────────────────────────────────────┘
```

- **역할:** 결론이 왜 나왔는지 신뢰할 수 있게 만든다.
- **있는 정보·요소:** 위험 점수, 섹터 판정, 이슈 지속성, 핵심 이슈
- **주 행동:** 근거 확인 후 포지션 관리로 이동

**핵심 가치:** 첫 화면부터 스탁가드가 "뉴스 요약"이 아니라 "내 보유 종목 기준 행동 판단"을 해준다는 점을 체감하게 한다.

**참고:** 첫 사용에서 종목을 최대 10개까지 받을 수 있지만, 입력 부담을 낮추기 위해 필수값은 종목명, 비중, 레버리지 여부로 제한한다.

## 플로우 2 · 포지션 등록과 원칙 설정: "내 종목은 어디서 위험해지나?"

**목표 순간:** 사용자가 자신의 주요 포지션과 위험 기준을 정리하고, 위험 조건이 닿았을 때 받을 알림과 매매 원칙을 확정한다.  
**상태 변화:** 막연히 들고 있는 종목 목록이, 장세와 가격 조건에 따라 행동 원칙이 붙은 관리 대상이 된다.

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주 행동 | 보조 |
|:--:|------|------|----------|------|
| 1 | 포지션 관리 화면 | 보유 종목 입력 및 검토 | 주요 종목 정보 보완 | 최대 10개 주요 종목 관리 |
| 2 | 포지션 관리 화면 | 섹터별 장세 판정 | 종목별 섹터와 위험도 확인 | 반도체, 2차전지 등 자동 추천 |
| 3 | 위험선/알림 설정 화면 | 가격 방어 알람 | 추천 위험선 확인 후 적용 | 전저점, 이동평균, 급락률 기준 |
| 4 | 매매 원칙 화면 | 개인 매매 원칙 리마인드 | 원칙 선택 또는 수정 | 레버리지 금지, 추가 매수 금지 등 |
| 5 | 위험선/알림 설정 화면 | 알림 연락처 설정 | 문자·이메일 수신 정보 입력 | 인증과 수신 동의 |

**화면**

### 1. 포지션 관리 화면

```
┌────────────────────────────────────┐
│ 내 주요 포지션                      │
│                                    │
│ 종목        비중   손익   유형      │
│ 삼성전자    30%   -4%    일반      │
│ SOXL        20%   -11%   레버리지   │
│ SK하이닉스  15%   +8%    일반      │
│                                    │
│ + 주요 종목 추가                    │
│ 3 / 10개 등록됨                     │
│                                    │
│ [위험선 추천 받기]                  │
└────────────────────────────────────┘
```

- **역할:** 사용자의 실제 위험 노출을 서비스가 이해하도록 만든다.
- **있는 정보·요소:** 종목, 비중, 손익률, 레버리지 여부, 섹터, 투자 기간
- **주 행동:** `[위험선 추천 받기]`
- **기본값:** 최대 10개는 "주요 종목" 기준이다. 모든 보유 종목을 완전히 대체하기보다, 실제 의사결정에 큰 영향을 주는 포지션을 우선 등록한다.

### 2. 위험선/알림 설정 화면

```
┌────────────────────────────────────┐
│ SOXL 위험선 추천                    │
│                                    │
│ 현재 장세: 변동장 3단계             │
│ 레버리지 적합도: 낮음               │
│                                    │
│ 추천 알림                           │
│ [ ] 전저점 이탈                     │
│ [ ] 장중 -5% 급락                   │
│ [ ] 나스닥 선물 급락 + 금리 상승    │
│ [ ] 반등 실패 후 재하락             │
│                                    │
│ 알림 문장                           │
│ 레버리지 축소 원칙에 해당합니다.    │
│ 손실 만회성 추가 매수는 하지 마세요.│
│                                    │
│ [적용하기]                          │
└────────────────────────────────────┘
```

- **역할:** 전문가용 조건 설정이 아니라, 사용자가 이해할 수 있는 위험선과 행동 문장으로 바꾼다.
- **있는 정보·요소:** 추천 위험선, 장세 단계, 레버리지 적합도, 알림 문장
- **주 행동:** `[적용하기]`

### 3. 매매 원칙 화면

```
┌────────────────────────────────────┐
│ 내 원칙                             │
│                                    │
│ [x] 하락장 2단계 이상에서는         │
│     레버리지 신규 매수 금지          │
│ [x] 급락 첫날에는 물타기 금지        │
│ [x] 손실 만회 목적의 대출 매수 금지  │
│ [ ] 장마감 전 갭하락 위험 확인      │
│                                    │
│ [저장하기]                          │
└────────────────────────────────────┘
```

- **역할:** 위험 상황에서 서비스가 사용자에게 어떤 문장으로 개입할지 정한다.
- **있는 정보·요소:** 원칙 목록, 선택 상태, 저장
- **주 행동:** `[저장하기]`

**핵심 가치:** 사용자가 자기 포지션을 입력하는 순간, 스탁가드는 단순 관심종목 앱이 아니라 "원칙 기반 리스크 관제"로 바뀐다.

## 플로우 3 · 장중 리스크 알림과 마감 전 판단: "장중에 흔들릴 때, 뭘 하지 말아야 하나?"

**목표 순간:** 사용자가 화면을 계속 보지 못해도 장중 위험 변화와 마감 전 다음날 위험을 놓치지 않고, 충동적 매매 대신 저장한 원칙을 다시 확인한다.  
**상태 변화:** 급락·반등·뉴스에 흔들리는 상태에서, 지금 해야 할 행동과 하지 말아야 할 행동이 분명한 상태로 바뀐다.

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주 행동 | 보조 |
|:--:|------|------|----------|------|
| 1 | 장중 알림 화면 | 장중 리스크 모니터링 | 위험 알림 확인 | 가격, 시장 지표, 섹터 조건 반영 |
| 2 | 장중 알림 화면 | 장세별 행동 코칭 | 행동·금지·이유 확인 | 초보자 실수 복기 포함 |
| 3 | 장중 시황 요약 화면 | 장중 문자 시황 | 시장 단계와 보유 종목 주의사항 확인 | 하루 3회 정도 |
| 4 | 마감 전 판단 화면 | 장마감 30분 전 다음날 예측 | 유지·축소·매도·현금화 판단 확인 | 다음날 갭 위험 반영 |
| 5 | 매매 원칙 화면 | 개인 매매 원칙 리마인드 | 원칙을 다시 확인하고 기록 | 나중에 복기 가능 |

**화면**

### 1. 장중 알림 화면

```
┌────────────────────────────────────┐
│ 위험 알림: SOXL                     │
│                                    │
│ 행동: 레버리지 비중 축소 원칙 해당  │
│ 금지: 손실 만회성 추가 매수 금지    │
│                                    │
│ 이유                               │
│ - 나스닥 선물 약세 확대             │
│ - 금리 변동성 상승                  │
│ - 장중 반등 실패 후 재하락          │
│                                    │
```

- **역할:** 알림을 단순 가격 도달 메시지가 아니라 행동 교정 메시지로 만든다.
- **있는 정보·요소:** 행동, 금지, 이유, 복기, 상세 보기, 원칙 확인
- **주 행동:** `[원칙 확인]` 또는 `[상세 보기]`

- **있는 정보·요소:** 시장 단계, 주요 변화, 내 종목 주의, 피할 행동
- **주 행동:** `[자세히 보기]`

## v1 설계 메모

- 첫 진입은 `주요 보유 종목 최대 10개`까지 허용하되, 필수 입력은 종목명, 비중, 레버리지 여부로 제한한다.
- 가격선, 손익률, 투자 기간, 알림 연락처는 첫 판단 이후 자연스럽게 보완하게 한다.
- 알림 문장은 투자 조언처럼 단정하기보다, 사용자가 저장한 원칙과 현재 위험 조건에 근거한 리스크 코칭으로 표현한다.
- 대시보드는 결론을 먼저 말하고, 근거 점수와 데이터는 바로 아래에서 확인하게 한다.
- 장중 알림은 모든 변화를 같은 강도로 보내지 않고, 레버리지·급락·금리 급등·섹터 급락처럼 손실을 크게 키울 조건을 우선한다.
