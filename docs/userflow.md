## 플로우 1 · 개장 전 방어 브리핑: "오늘 장, 일단 줄여야 하나?"

**목표 순간:** 출근 직후 3분 안에 오늘 한국장과 내 주요 보유 종목 기준으로 `방어 우선`, `유지 가능`, `신규 매수 금지`, `레버리지 부적합` 같은 결론을 얻는다.  
**상태 변화:** 정보가 흩어져 불안한 상태에서, 오늘의 대응 우선순위와 하지 말아야 할 행동이 분명한 상태로 바뀐다.

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주 행동 | 보조 |
|:--:|------|------|----------|------|
| 1 | 환영/빠른 등록 화면 | 보유 종목 입력 및 검토 | 주요 보유 종목 1~10개 입력 | 종목명, 비중, 레버리지 여부만 먼저 받음 |
| 2 | 환영/빠른 등록 화면 | 알림 연락처 설정 | 나중에 설정 또는 간단 등록 선택 | 첫 진입에서는 건너뛰기 가능 |
| 3 | 아침 브리핑 대시보드 | 개장 전 AI 종합 브리핑 | 오늘의 종합 판단 확인 | 결론을 맨 위에 표시 |
| 4 | 아침 브리핑 대시보드 | 명확한 행동 지침 | 해야 할 행동과 금지 행동 확인 | 방어/유지/축소/금지 라벨 |
| 5 | 근거 상세 화면 | 장세·섹터 판정 | 근거 점수와 주요 이슈 확인 | 금리, 환율, 유가, 미국장, 수급, 섹터 점수 |
| 6 | 포지션 관리 화면 | 보유 종목 검토 | 더 자세한 종목 정보 보완 | 가격선, 손익률, 투자 기간은 이후 입력 |

**화면**

### 1. 환영/빠른 등록 화면

```
┌────────────────────────────────────┐
│ 스탁가드                           │
│ 오늘 장에서 무엇을 줄이고          │
│ 무엇을 하지 말아야 할지 먼저 봅니다 │
│                                    │
│ 주요 보유 종목                     │
│ [ 삼성전자     ][비중 30%][일반]   │
│ [ SOXL         ][비중 20%][레버리지]│
│ [ SK하이닉스   ][비중 15%][일반]   │
│ + 종목 추가                       │
│                                    │
│ 최대 10개까지 입력 가능            │
│                                    │
│             [오늘 판단 보기]       │
└────────────────────────────────────┘
```

- **역할:** 첫 사용자가 복잡한 가입 절차보다 먼저 개인화 판단을 받을 수 있게 한다.
- **있는 정보·요소:** 종목명, 보유 비중, 레버리지 여부, 종목 추가, 오늘 판단 보기
- **주 행동:** `[오늘 판단 보기]` 선택
- **기본값:** 주요 보유 종목은 최대 10개까지 허용한다. 손익률, 현재가, 매수가는 나중에 보완해도 된다.

### 2. 아침 브리핑 대시보드

```
┌────────────────────────────────────┐
│ AI 종합 판단: 오늘은 방어 우선      │
│ 현재 장세: 변동장 3단계             │
│ 하락 리스크: 78점                  │
│                                    │
│ 해야 할 행동                       │
│ - 레버리지 비중 축소 검토           │
│ - 반도체 비중 신규 확대 보류        │
│                                    │
│ 하지 말아야 할 행동                 │
│ - 손실 만회성 추가 매수 금지        │
│ - 장초반 반등만 보고 추격 매수 금지 │
│                                    │
│ [내 종목 영향 보기] [근거 보기]     │
└────────────────────────────────────┘
```

- **역할:** 데이터보다 결론을 먼저 보여준다.
- **있는 정보·요소:** AI 종합 판단, 장세 단계, 리스크 점수, 행동, 금지, 내 종목 영향, 근거 보기
- **주 행동:** `[내 종목 영향 보기]` 또는 `[근거 보기]`

### 3. 근거 상세 화면

```
┌────────────────────────────────────┐
│ 판단 근거                           │
│ 금리 위험        82  높음           │
│ 환율 위험        71  높음           │
│ 유가 위험        68  주의           │
│ 반도체 섹터      84  취약           │
│ 이슈 지속성      높음               │
│ 수급 위험        76  외국인 매도 우위│
│ 채권 이동        72  방어 자금 유입  │
│                                    │
│ 큰 장세 × 오늘 상황                 │
│ 큰 장세: 전쟁 리스크 확대           │
│ 오늘 상황: 휴전 기대 단기 완화      │
│                                    │
│        오늘 상황                    │
│   완화 ┌──────────────┐ 악화        │
│        │   ● 단기 완화 │             │
│        │              │             │
│        │ 큰 압력은 아직│             │
│        └──────────────┘             │
│       낮음      큰 장세 압력     높음│
│                                    │
│ 수급                                   │
│ - 코스피 외국인: 순매도 확대        │
│ - 기관: 방어 업종 매수, 성장주 매도 │
│ - 개인: 거래량 증가, 저가 매수 유입 │
│ - 내 종목: 반도체 대형주 매도 우위  │
│ - 대형 IPO/테마: 성장주 자금 흡수 가능│
│ - 채권: 가격 상승, 거래량 증가       │
│                                    │
│ 핵심 이슈                         │
│ - 미국 기술주 약세                 │
│ - 금리 변동성 확대                 │
│ - 반도체 지수 시장 대비 부진        │
└────────────────────────────────────┘
```

- **역할:** 결론이 왜 나왔는지 신뢰할 수 있게 만든다.
- **있는 정보·요소:** 위험 점수, 섹터 판정, 이슈 지속성, 큰 장세·오늘 상황 2차원 맵, 외국인·기관·개인 수급, 대형 자금 이동 이벤트, 채권 가격·거래량·자금 이동, 핵심 이슈
- **주 행동:** 근거 확인 후 포지션 관리로 이동

**핵심 가치:** 첫 화면부터 스탁가드가 "뉴스 요약"이 아니라 "내 보유 종목 기준 행동 판단"을 해준다는 점을 체감하게 한다.

**참고:** 첫 사용에서 종목을 최대 10개까지 받을 수 있지만, 입력 부담을 낮추기 위해 필수값은 종목명, 비중, 레버리지 여부로 제한한다.

## 플로우 2 · 포지션 등록과 원칙 설정: "내 종목은 어디서 위험해지나?"

**목표 순간:** 사용자가 자신의 주요 포지션과 위험 기준을 정리하고, 위험 조건이 닿았을 때 받을 알림과 매매 원칙을 확정한다.  
**상태 변화:** 막연히 들고 있는 종목 목록이, 장세와 가격 조건에 따라 행동 원칙이 붙은 관리 대상이 된다.

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주 행동 | 보조 |
|:--:|------|------|----------|------|
| 1 | 포지션 관리 화면 | 보유 종목 입력 및 검토 | 주요 종목 정보 보완 | 최대 10개 주요 종목 관리 |
| 2 | 포지션 관리 화면 | 섹터별 장세 판정 | 종목별 섹터와 위험도 확인 | 반도체, 2차전지 등 자동 추천 |
| 3 | 위험선/알림 설정 화면 | 가격 방어 알람 | 추천 위험선 확인 후 적용 | 전저점, 이동평균, 급락률 기준 |
| 4 | 매매 원칙 화면 | 개인 매매 원칙 리마인드 | 원칙 선택 또는 수정 | 레버리지 금지, 추가 매수 금지 등 |
| 5 | 위험선/알림 설정 화면 | 알림 연락처 설정 | 문자·이메일 수신 정보 입력 | 인증과 수신 동의 |

**화면**

### 1. 포지션 관리 화면

```
┌────────────────────────────────────┐
│ 내 주요 포지션                      │
│                                    │
│ 종목        비중   손익   유형      │
│ 삼성전자    30%   -4%    일반      │
│ SOXL        20%   -11%   레버리지   │
│ SK하이닉스  15%   +8%    일반      │
│                                    │
│ + 주요 종목 추가                    │
│ 3 / 10개 등록됨                     │
│                                    │
│ [위험선 추천 받기]                  │
└────────────────────────────────────┘
```

- **역할:** 사용자의 실제 위험 노출을 서비스가 이해하도록 만든다.
- **있는 정보·요소:** 종목, 비중, 손익률, 레버리지 여부, 섹터, 투자 기간
- **주 행동:** `[위험선 추천 받기]`
- **기본값:** 최대 10개는 "주요 종목" 기준이다. 모든 보유 종목을 완전히 대체하기보다, 실제 의사결정에 큰 영향을 주는 포지션을 우선 등록한다.

### 2. 위험선/알림 설정 화면

```
┌────────────────────────────────────┐
│ SOXL 위험선 추천                    │
│                                    │
│ 현재 장세: 변동장 3단계             │
│ 레버리지 적합도: 낮음               │
│                                    │
│ 추천 알림                           │
│ [ ] 전저점 이탈                     │
│ [ ] 장중 -5% 급락                   │
│ [ ] 나스닥 선물 급락 + 금리 상승    │
│ [ ] 반등 실패 후 재하락             │
│                                    │
│ 알림 문장                           │
│ 레버리지 축소 원칙에 해당합니다.    │
│ 손실 만회성 추가 매수는 하지 마세요.│
│                                    │
│ [적용하기]                          │
└────────────────────────────────────┘
```

- **역할:** 전문가용 조건 설정이 아니라, 사용자가 이해할 수 있는 위험선과 행동 문장으로 바꾼다.
- **있는 정보·요소:** 추천 위험선, 장세 단계, 레버리지 적합도, 알림 문장
- **주 행동:** `[적용하기]`

### 3. 매매 원칙 화면

```
┌────────────────────────────────────┐
│ 내 원칙                             │
│                                    │
│ [x] 하락장 2단계 이상에서는         │
│     레버리지 신규 매수 금지          │
│ [x] 급락 첫날에는 물타기 금지        │
│ [x] 손실 만회 목적의 대출 매수 금지  │
│ [ ] 장마감 전 갭하락 위험 확인      │
│                                    │
│ [저장하기]                          │
└────────────────────────────────────┘
```

- **역할:** 위험 상황에서 서비스가 사용자에게 어떤 문장으로 개입할지 정한다.
- **있는 정보·요소:** 원칙 목록, 선택 상태, 저장
- **주 행동:** `[저장하기]`

## v1 설계 메모

- 첫 진입은 `주요 보유 종목 최대 10개`까지 허용하되, 필수 입력은 종목명, 비중, 레버리지 여부로 제한한다.
- 가격선, 손익률, 투자 기간, 알림 연락처는 첫 판단 이후 자연스럽게 보완하게 한다.
- 장중에 사용자가 기존 주식을 매도하거나 추가 매수하거나 새로운 주식을 매수하면, 최신 포지션 기준으로 위험도와 행동 지침을 다시 계산한다.
- v1에서는 장중 체결 내용을 사용자가 직접 입력하는 방식을 기본으로 하고, 이후 증권 계좌·체결 내역 연동을 확장 후보로 둔다.
- 장마감 전 브리핑은 발표가 끝난 뒤의 사후 해석이 아니라, 발표 전 예상 범위와 결과별 시장 영향 시나리오를 포함한다.
- 사용자가 묻지 않아도 오늘 밤과 다음날 새벽 예정된 미국 경제지표, 연준 일정, 주요 기업 실적을 자동으로 스캔해 마감 전 판단에 반영한다.
- 반도체 관련 보유 종목이나 섹터 노출이 있으면 NVIDIA, AMD, Intel, Micron, Broadcom, ASML, TSMC, Applied Materials 같은 주요 반도체·AI 인프라 기업의 실적과 가이던스를 별도 중요 이벤트로 다룬다.
- 사용자가 "오늘 밤 10시 30분 비농업고용지표가 어떻게 나올 것 같고, 결과별로 시장은 어떻게 움직일까?"처럼 질문하면, 해당 이벤트를 마감 전 판단에 더 자세히 반영한다.
- 지표 예상은 단일 숫자 단정보다 컨센서스, 최근 선행지표, 시장 포지셔닝, 금리·환율 반응을 함께 본 예상 범위와 시나리오로 표현한다.
- 실적 영향은 매출·EPS 결과뿐 아니라 다음 분기 가이던스, AI 수요, 메모리 가격, 마진, 재고, 설비투자 코멘트가 보유 섹터와 국내 종목에 미칠 영향을 함께 본다.
- 현재 장세가 약세장이나 변동장일수록 같은 부정적 지표·실적도 더 크게 반영하고, 긍정적 결과는 단기 반등인지 추세 전환인지 분리해 판단한다.
- 전쟁, 정책 충돌, 공급망 리스크처럼 큰 장세 압력과 휴전, 충돌 완화, 단기 호재 같은 오늘의 작은 상황은 2차원으로 분리해 보여준다.
- 2차원 맵은 `큰 장세 압력 낮음↔높음`과 `오늘 상황 완화↔악화` 축으로 표현하고, 현재 위치가 "큰 압력 안의 단기 완화"인지 "큰 압력과 단기 악화가 겹친 위험 구간"인지 판단한다.
- 수급은 코스피 외국인 순매수·순매도, 기관 수급, 개인 거래량과 사용자가 입력한 종목별 외국인·기관·개인 수급을 함께 본다.
- SpaceX 같은 대형 IPO, 초대형 증자, 대형 테마 상장, 특정 인기 종목으로 자금이 몰리는 이벤트는 기존 성장주·기술주·반도체 주식의 수급을 빼앗을 수 있는 자금 흡수 이벤트로 반영한다.
- 대형 자금 이동 이벤트는 실제 상장 여부만 보는 것이 아니라, 상장 기대, 청약 대기 자금, 기존 주식 매도 후 현금 확보 흐름, 관련 테마 쏠림까지 함께 본다.
- 채권은 금리만 보지 않고 채권 가격, 거래량, 채권 ETF 자금 유입·유출, 장단기 국채 수요, 주식에서 채권으로 이동하는 방어적 자금 흐름을 함께 본다.
- 채권 가격 상승과 거래량 증가는 위험 회피 자금 이동일 수 있으므로, 주식시장 수급 약화와 함께 나타나면 방어 신호로 가중한다.
- 개인 거래량 급증은 항상 호재로 보지 않고, 약세장에서 저가 매수 쏠림인지 실제 추세 회복인지 구분한다.
- 알림 문장은 투자 조언처럼 단정하기보다, 사용자가 저장한 원칙과 현재 위험 조건에 근거한 리스크 코칭으로 표현한다.
- 대시보드는 결론을 먼저 말하고, 근거 점수와 데이터는 바로 아래에서 확인하게 한다.
- 장중 알림은 모든 변화를 같은 강도로 보내지 않고, 레버리지·급락·금리 급등·섹터 급락처럼 손실을 크게 키울 조건을 우선한다.
