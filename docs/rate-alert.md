# 미국 2년물 금리 급등락 문자 알람 (rate-alert) — v1.0 (2026-07-06)

## 1. 배경·목적

2026-06-05, 06-18, 06-30, 07-01 — 미국 2년물 금리가 급상승한 날 주가지수가 큰 폭으로
하락했다. 2년물 금리의 **급변(30분/1시간 변동)** 과 **절대 레벨(4.125% 돌파)** 을 무인
모니터링해서, 매도(금리 급등)·매수(금리 급락) 검토 시점을 문자로 알린다.
추가로 10년물 금리의 위험 레벨(4.45%) 돌파도 함께 감시한다.

## 2. 실측 분석 (scripts/rate-alert-analyze.ts)

야후에는 2년물 금리 분봉이 없어(US2YT=X 미존재, 2YY=F는 유동성 제로) CME 2년물
국채선물 **ZT=F 30분봉**을 금리로 환산했다. 환산계수는 네이버 실시간 금리(US2YT=RR)
일봉과의 회귀로 실측: **Δyield ≈ -0.475 %p/pt (R²=0.875, 26일 표본)**.

| 날짜 | 일봉 (시→종, 고/저) | \|30분\| 최대 | \|1시간\| 최대 | 특징 |
|---|---|---|---|---|
| 06-05 | 4.047→4.162, 고 4.178 | **0.100%p** (08:30 ET) | 0.102%p | 고용지표 서프라이즈 — 전형적 급등 |
| 06-18 | 4.180→4.179, 고 4.217 | 0.020%p | 0.033%p | 완만형 — 고점이 4.125 크게 상회 |
| 06-30 | 4.106→4.139, 고 4.178 | 0.013%p | 0.022%p | 지속 상승형 — 장중 4.125 상향 돌파 |
| 07-01 | 4.170→4.164, 고 4.199 | 0.032%p | 0.039%p | 급변+레벨 둘 다 해당 |

평상시(이벤트 4일 제외, 5/25~7/6) 30분 변동 분포: 중앙값 0.002 · 95% 0.011 · **99% 0.022**
· 최대 0.070%p. 1시간: 95% 0.017 · 99% 0.039 · 최대 0.093%p.

**임계값 시뮬레이션 결과:**

- ±0.02%p/30분: 이벤트 3/4일 감지, 평시 10일 발동(과다)
- **±0.03%p/30분: 이벤트 2/4일(06-05·07-01), 평시 5일** ← 기본값 채택
- **±0.03%p/1시간: 이벤트 3/4일(06-05·06-18·07-01), 평시 5일** ← 기본값 채택
- 나머지 완만형(06-30)은 **레벨 조건(4.125 돌파)** 이 커버 → 조합하면 **4/4일 전부 감지**

평시 발동일(06-11·06-17·06-25·07-02 등)도 실제 금리 변동이 컸던 날이라 순수 오탐이
아니며, 알림 키별 1일 1회 중복 방지로 하루 최대 문자 수는 제한된다.

## 3. 알람 조건 (모두 OR, 키별 1일 1회)

| 키 | 조건 | 방향 | 문자 |
|---|---|---|---|
| `rate2y_spike_up` | 30분 변동 ≥ +Δ30 또는 1시간 변동 ≥ +Δ1h | 급등 | 매도 검토 |
| `rate2y_spike_down` | 30분 변동 ≤ −Δ30 또는 1시간 변동 ≤ −Δ1h | 급락 | 매수 검토 |
| `rate2y_lvl_u<레벨>` | 2년물이 단계 레벨 **상향 돌파** — 4.135 경고 / 4.15 위험 / 4.16 최고위험 | 완만 상승 | 경고=주식 1/3 감축 · 위험=2/3 매도 · 최고위험=전량 매도 검토 |
| `rate2y_lvl_d<레벨>` | 2년물이 단계 레벨 **하향 이탈** | 완만 하락 | 단계 해제 |
| `rate10y_level_up` | 10년물이 레벨(4.45) 상향 돌파 | 위험 | 고PER 압박 경고 |

- (개정 2026-07-07) 원래 4.125 단일 기준이었으나 금리가 4.12X에서 진동하며 돌파/이탈
  문자가 반복되는 노이즈 발생 → 사용자 요청으로 제거하고 3단계 레벨로 교체.
- 단계 판정은 상태 기계: 직전 샘플의 단계와 현재 단계를 비교해 올라가면 돌파, 내려오면
  해제 알림. 한 번에 여러 단계를 건너뛰면 최종 단계 1건만 발송. 첫 가동(직전 샘플 없음)
  시 현재가 단계 위면 그 단계 1회 알림.
- 급변 조건의 변동값은 '현재 샘플 vs 30분(±10분)/60분(±15분) 전 샘플' 차이로 계산.

## 5. 아키텍처

- **데이터 소스**: 네이버 채권 API (실시간·무지연)
  - 2년물: `api.stock.naver.com/marketindex/bond/US2YT=RR`
  - 10년물: `api.stock.naver.com/marketindex/bond/US10YT=RR`
  - FRED(DGS2)는 1~2영업일 지연이라 부적합, 야후 ^TNX는 10년물 폴백으로만.
- **샘플 축적**: `rate_samples` 테이블(ts, y2, y10) — 크론 호출마다 1행 저장,
  7일 지난 행은 자동 삭제. 30분/1시간 변동은 저장된 샘플 간 차이로 계산하므로
  분봉 API 의존이 없다.
- **판정·발송**: `/api/cron/rate-alert?secret=<CRON_SECRET>` 호출 시
  샘플 저장 → 조건 판정 → 문자(Solapi)+이메일(Resend) 발송 → `alerts`(trigger_key='rate')
  이력 기록. 수신자는 기존 `alert_channels`(인증+동의) 그대로.
- **스케줄**: cron-job.org에 **10분 간격** 등록 (사용자 직접 등록 — signal state와 동일 방식).
  10분 간격이어야 30분 변동을 ±10분 정확도로 잡는다. 미국채는 도쿄~런던~뉴욕 사이
  거의 24시간 거래되므로 종일 감시(주말·휴장엔 값이 안 변해 자연히 조용).
- **문자 형식**: 90바이트 이내 단문(SMS 요금) — 예:
  `[스탁가드] 미2년 금리급등 +0.04%p/30분 4.17% — 매도 검토`

## 6. 구현 파일

- `lib/market/rateAlert.ts` — 시세 조회 + 판정(순수 함수) + 설정
- `lib/alerts/dispatch.ts` — 공용 발송(1일 1회 중복 방지) — 신호(M7) 알림과 공유
- `app/api/cron/rate-alert/route.ts` — 크론 엔드포인트
- `supabase/migrations/018_rate_alert.sql` — rate_samples + trigger_key 'rate' 허용
- `scripts/rate-alert-analyze.ts` — 임계값 근거 분석 (데이터 창이 지나면 EVENT_DAYS·PERIOD를 바꿔 재분석)
- `scripts/rate-alert-test.ts` — 판정 로직 회귀 검증 14종 (로직 수정 후 반드시 실행)

### 화면 (시장·알림·분석 > 장중 시황 요약 > 매크로·금리)

- `lib/market/rateIntraday.ts` — 30분 시계열 조립: 실측(rate_samples) 버킷 우선,
  빈 구간은 ZT=F 30분봉 환산(계수 -0.475, 현재 실측 금리에 앵커링)으로 보강.
  주말·휴장 직후에도 보이도록 벽시계 창이 아니라 '최근 60개 봉' 기준.
- `app/market/intraday/_client.tsx`의 `Us2yCard` — 채권 동향 카드 아래 배치.
  현재값·전일 대비·기준선(4.125) 대비 배지 + 30분 그래프(기준선 점선) +
  30분 단위 표(시각 KST·금리·30분/1시간 변동·출처, 임계값 돌파 구간 ⚠ 강조) + 알람 조건 요약.

## 7. 가동 절차 (남은 수동 단계)

1. Supabase Dashboard SQL Editor에서 `supabase/migrations/018_rate_alert.sql` 실행
   (미적용 시 크론이 500 "샘플 저장 실패"로 응답 — 문자 오발송 위험은 없음).
2. cron-job.org에 등록: `https://test-project-0613.vercel.app/api/cron/rate-alert?secret=<CRON_SECRET>`
   — **10분 간격, 매일** (미국채는 거의 24시간 거래. 주말·휴장엔 값이 안 변해 자연히 조용).
3. 임계값 조정 시 `.env.local`과 Vercel 환경변수 양쪽에 `RATE_ALERT_*` 추가 후 재배포.
4. 첫 가동 시 현재 상태 기준 알림이 올 수 있음 — 현재 2년물 4.137%(기준 4.125 위),
   10년물 4.485%(기준 4.45 위)라 두 건의 '기준선 위' 알림이 1회 발송된다.
